# 1. 문제
소문자로 된 단어(문자열)가 입력되면,  
그 단어의 가운데 문자를 반환하시오. 단어가 짝수일 경우에는 가운데 2개의 문자를 출력한다.  

입력예제: "study"  
출력예제: 'u'  

입력예제: "studys"  
출력예제: 'ud'  

# 2. 풀이
```js
function solution (words) {
  let answer;
  let mid = Math.floor(s.length / 2);

  if (words.length % 2 == 1) answer = words.substring(mid, mid + 1);
  else answer = words.substring(mid - 1, mid + 1);

  return answer;
}

// const words = "studys";
const words = "HelloWorld";

solution(words);
```

주어진 단어의 길이의 중간을, <span style="color:blue">변수 mid</span>에 넣어서 관리하는 것이 가독성과 유지보수 측면에서 더 좋다.  
그리고 나의 풀이에서는 string 을 array 로 만든 다음에 ->  
여기서 splice를 사용하고 있는데 이것보다도  
<span style="color:tomato">substring 메서드</span>를 사용하여 <u>바로 반환하는 것이 훨씬 보기 좋아 보인다</u>.  

# 3. 나의 풀이
```js
function solution (words) {
  if ((words.length) % 2 !== 0) 
    return words[Math.floor(words.length / 2)]
  else return words.split('').splice((words.length / 2) - 1, 2);
}

// const words = "studys";
const words = "HelloWorld";

solution(words);
```
