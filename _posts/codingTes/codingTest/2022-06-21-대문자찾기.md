# 1. 문제
한 개의 문자열을 입력받아 해당 문자열에 알파벳 대문자가 몇 개 있는지 작성하여라.  

입력예제: KoreaTimeGood  

출력예제: 3  

# 2. 풀이
```js
function solution (words) {
  let result = 0;
  
  for (word of words) {
    if (word === word.toUpperCase()) result += 1;
  }
  return result;
}

const words = 'KoreaTimeGood';

solution(words);
```

`toUpperCase()`를 사용하여 word에 담긴 element를 대문자로 바꾸어 비교를 실시한다.  

# 3. 나의 풀이
```js
function solution (word) {
  let result = word.split('').filter((ele, ind) => {
    if (ele.charCodeAt(0) <= 90) return true;
  });
  
  return result.length;
}


const word = 'KoreaTimeGood';

solution(word);
```
