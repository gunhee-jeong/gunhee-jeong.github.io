# 1. 문제
N명의 학생의 국어점수가 입력되면,  
각 학생의 등수를 입력된 순서대로 반환하시오.  

같은 점수가 입력된 경우 높은 등수로 동일 처리한다.  
가장 높은 점수가 92점일 경우, 동점자가 3명 존재한다면 1등은 3명이다.  
그리고 다음 학생은 4등이 된다.  

[입력예제]  
[87, 89, 92, 100, 76]  
[87, 89, 92, 100, 100]  

[출력예제]  
[4, 3, 2, 1, 5]  
[5, 4, 3, 1, 1]  

# 2. 풀이
```js
function solution (arr) {
  let n = arr.length;
  let result = Array.from({length: n}, () => 1);
  for (let i = 0; i < n; i++>) {
    for (j = 0; j < n; j++>) {
      if (arr[j] > arr[i]) result[i]++;
    }
  }
  return result;
}

const arr = [87, 89, 92, 100, 100];

solution(arr);
```

# 3. 나의 풀이
```js
function solution (arr) {
  let result = [];
	let sortList = [...arr];
  
  sortList.sort((a, b) => b - a);

	arr.forEach((ele, ind) => {
		result.push(sortList.indexOf(ele) + 1);
  });
  return result;
}

const arr = [87, 89, 92, 100, 100];

solution(arr);
```
