# 🔴 풀이
```jsx
function solution(s) {
  const stack = [];
  
  for (let i = 0; i < s.length; i++) {
    if (stack.length === 0) { 
      stack.push(s[i]);
      continue;
    }
    if (stack[stack.length - 1] === s[i]) { 
      stack.pop();
      continue;
    }
    stack.push(s[i]);
  }
  
  return stack.length === 0 ? 1 : 0;
} 
```

처음에는 `for`문을 두 번 돌면서 효율성 테스트를 통과하지 못했었다. 그래서 힌트를 얻어보니 stack 구조를 사용할 수 있었다. `stack`이라는 변수를 하나 만들어주고 `s`의 string을 하나씩 `push`하고 비교하면서 문제를 해결하였다.
