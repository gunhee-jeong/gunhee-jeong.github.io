# 문제설명

## 설명

함수 solution은 정수 x와 자연수 n을 입력 받아, x부터 시작해 x씩 증가하는 숫자를 n개 지니는  
리스트를 리턴해야 합니다. 다음 제한 조건을 보고, 조건을 만족하는 함수, solution을 완성해주세요.

## 입출력 예

|  x  |  n  |         answer         |
| :-: | :-: | :--------------------: |
|  2  |  5  | [5, 6, 3] [2,4,6,8,10] |
|  4  |  3  |        [4,8,12]        |
| -4  |  2  |        [-4, -8]        |

## 주어진 코드

```java
let x = 4;
let n = 3;

function solution(x, n) {

}

console.log(solution(x, n));
```

# 문제풀이

## 나의 풀이

```java
let x = 2;
let n = 5;

function solution(x, n) {
  let add = n;
  let result = [];
  for(let i = 1; i < n + 1; i++) {
    result.push(x * i)
  }

  return result;
}

console.log(solution(x, n));
```

## 좋아요가 가장 많은 풀이

```java
let x = 2;
let n = 5;
//return시 [2,4,6,8,10]을 반환함

function solution(x, n) {
    return Array(n).fill(x).map((v, i) => (i + 1) * v)
}

console.log(solution(x, n));
```

Array(n)

- `Array()`를 사용하여 <u>[empty * 5]를 생성</u>한다.

Array(n).fill(x)

- `.fill()`를 사용하여 위에서 <u>생성된 array의 element로 모두 2가 채워지도록</u> 설정함 →
  **[2, 2, 2, 2, 2]**

Array(n).fill(x).map((v, i) => (i + 1) \* v)

- **`.map()`**의 함수의 **parameter**로 value와 index를 받고 **index에 +1**을 하여서  
  2 \* 1, 2 \* 2, 2 \* 3으로 array의 element를 채워나가도록 설정함
