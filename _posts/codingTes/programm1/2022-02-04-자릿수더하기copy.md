# 문제설명

- 자연수 N이 주어지면, N의 각 자릿수의 합을 구해서 return 하는 solution 함수를 만들어 주세요.  
  예를들어 N = 123이면 1 + 2 + 3 = 6을 return 하면 됩니다.
- |  N  | answer |
  | :-: | :----: |
  | 123 |   6    |
  | 987 |   24   |
- 주어진 코드

  ```javascript
  let n = 123;

  function solution(n) {}

  solution(n);
  ```

# 문제풀이

- 나의 풀이

  ```java
  let n = 123;

  function solution(n) {
    return (n + "").split("").reduce((acc, val) => acc + Number(val), 0);
  }

  console.log(solution(n)); //output == 6
  ```

  - <u>n의 값은 'number 타입'</u>으로 -> 이를 **string 타입으로 바꾸기 위해** <span style="color:blue">n + ""</span>을 사용했다  
    이로인해 123은 -> '123'의 string이 되었고 split method를 사용할 수 있게 되었다!  
    결과적으로 <u>n의 값이 변하는 과정</u>을 보자면, <span style="color:red">123 -> '123' -> ['1', '2', '3']</span>
  - 그리고 1 + 2 + 3, 이렇게 누적해서 6을 만들어야 하기 때문에 <span style="color:red">reduce</span> method를 사용해보았다
  - **val(value)에는 '1' -> '2' -> '3'**이 순서대로 들어가며 <span style="color:blue">number 타입으로</span> 바뀌어야 누적해서 더할 수  
    있기 때문에 **Number(val)**를 사용하여 string -> number로 바꾸는 과정을 추가했다!
