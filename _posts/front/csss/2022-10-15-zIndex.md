# 모달창과 z-index
(MDN) : [z-index](https://developer.mozilla.org/ko/docs/Web/CSS/z-index)

`z-index` 속성은 position 속성과 그 자손 또는 하위 아이템의 <u>z축 순서</u>를 지정한다. 더 큰 z-index 값을 가진 요소가 작은 값의 요소 위를 덮게 된다.

```html
<div class="dashed-box">Dashed box
  <span class="gold-box">Gold box</span>
  <span class="green-box">Green box</span>
</div>
```

```css
.dashed-box {
  position: relative;
  z-index: 1;
  border: dashed;
  height: 8em;
  margin-bottom: 1em;
  margin-top: 2em;
}
.gold-box {
  position: absolute;
  z-index: 3; /* put .gold-box above .green-box and .dashed-box */
  background: gold;
  width: 80%;
  left: 60px;
  top: 3em;
}
.green-box {
  position: absolute;
  z-index: 2; /* put .green-box above .dashed-box */
  background: lightgreen;
  width: 20%;
  left: 65%;
  top: -25px;
  height: 7em;
  opacity: 0.9;
}
```
