# 리액트 에러 모음
## 1. ReactDOM.render is no longer supported in React 18. Use createRoot instead.

<img src="https://user-images.githubusercontent.com/87808288/188276634-b4d4426b-d476-4492-ab29-5c205e0e5989.png" width="70%">  

리액트를 18 버전으로 업그레이드한 이후에 테스트 코드를 작성하면서 위와 같은 메시지의 에러가 발생하였다. 이때에는 react testing library 또한 버전을 업데이트 해주어야한다. 방법은 아래와 같이 순서대로 터미널에서 명령어를 입력하면 된다.

```bash
npm install --save-dev @testing-library/react@latest
npm install --save-dev @testing-library/jest-dom@latest
npm install --save-dev @testing-library/user-event@latest
```
