#key 이벤트의 종류

## keydown

키보드를 <span style="color:red">누를 때</span> 실행!  
키를 누르고 있을 때, <u>단 한 번만 실행</u>된다.

그리고 keydown은 `모든 문자를 인식`한다.  
keydown은 키보드의 어떤 키가 눌러지던지 <span style="color:red">모든 key에 반응</span>한다.

## keypress

키보드를 <span style="color:red">누를 때</span> 실행  
키를 누를 때마다 <u>계속 실행</u>된다.

그리고 keypress는 `한글을 지원하지 못한다`.  
`한글 입력, 방향키, delete키`와 같은 <span style="color:red">text에 반영되지 않는 key</span>에는 <u>동작하지 않는다</u>.

## keyup

누른 키에서 <span style="color:red">손을 뗄 때</span> 실행된다.

# key 이벤트의 동작 순서

키를 누르면 keydown 이벤트가 진행되는데, 그 진행의 순서는  
`keydown` -> `keypress` -> `keyup`의 순으로 진행된다.  
(키를 누르면 keydown이 동작하고 -> 바로 keypress가 동작 -> 키가 해제되면 keyup이 동작한다)
