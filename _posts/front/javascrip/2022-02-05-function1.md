```java
// doubleNumber 함수 선언식
function doubleNumber(myNumber) { // <- 호출시 전달받을 값을 myNumber 라는 변수명으로 함수 안쪽에서 사용하겠다.
  let myResult = myNumber * 2
  return myResult
}

// doubleNumber 함수 호출문
doubleNumber(3); // <- doubleNumber를 호출하면서 값 3을 전달해주겠다.
```

# parameter

<u>functiton</u>을 정의하면서, 함수 선언식의 `괄호()안에 어떤 변수명`을 사용하면 -> 그것을 <span style="color:red">parameter</span>라고 부른다

- parameter는 실제로 <span style="color:red">함수 안쪽에서 변수와 같은 역할</span>을 하게 된다!  
  <u>함수가 호출될 때 값을 전달받으면</u> `parameter에 값이 정의`된다
- 위의 코드, function doubleNumber<span style="color:green">(myNumber)</span> {에서  
  <span style="color:red">myNumber</span>에는 <u>함수 호출시 전달받은 값을</u> **myNumber라는 변수명으로**, 함수 안쪽에서 사용하겠다는 것이다!

# argument

- <u>function을 호출</u> `호출문의 괄호 안에 어떤 값 또는 정의된 변수`를 사용하면  
  이것을 <span style="color:red">argument</span>라고 부른다
