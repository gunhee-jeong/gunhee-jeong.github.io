# 'push' and 'concat'
# 🔴 push
`push` 메서드는 배열의 <span class="cf">마지막 부분에 element를 추가</span>한다. `push`는 인자로 전달받은 모든 값을 <span class="cf">원본 배열의 마지막 element로 추가</span>하고 변경된 <span class="cf">length 값을 반환</span>한다. `push` 메서드는 원본 배열을 직접 변경한다.

```jsx
const fruits = ['🍎', '🍌'];
let addFruits = fruits.push('🍓', '🍑');

console.log(addFruits); //output == 4
console.log(fruits); //outcome == (4) ['🍎', '🍌', '🍓', '🍑']
```

# 🔴 concat
`push` 메서드와 `concat` 메서드는 유사하게 동작하지만 미묘한 차이가 발생한다.

`push` 메서드는 원본 배열을 직접 변경하지만, `concat` 메서드는 <span class="cf">원본 배열을 변경하지 않고</span> 새로운 배열을 반환한다.

인자로 전달받은 값이 배열인 경우 `push` 메서드는 배열을 그대로 원본 배열의 마지막 요소에 추가하지만 `concat` 메서드는 <span class="cf">배열을 해체</span>하여 새로운 배열의 마지막 element로 추가한다.

```jsx
const arr1 = [1, 2];

arr1.push([3, 4]);
console.log(arr1); // [1, 2, [3, 4]]

const arr2 = [1, 2];
// 인수로 전달받은 배열을 해체하여 새로운 배열의 마지막 요소로 추가한다
const result = arr2.concat([3, 4]);
console.log(result); // [1, 2, 3, 4]
```
