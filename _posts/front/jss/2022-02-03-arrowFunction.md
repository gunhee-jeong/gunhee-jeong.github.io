# arrow function의 소괄호

- **arrow function**의 경우 <u>argument가 하나</i>일 때는 <span style="color:red">소괄호의 생략</span>가능!

  ```javascript
  //ES5
  const getName = function (name) {};

  //ES6
  const getName = name => ;
  ```

  argument가 두 개일 때는 생략할 수 없다

  ```javascript
  //ES5
  const getName = function (name, age) {};

  //ES6
  const getName = (name, age) => {};
  ```

# arrow function의 return

- 함수의 <u>실행내용이 return만</u> 있다면 **return 키워드와 중괄호도 생략**이 가능!

  ```javascript
  //ES5
  function hi(text) {
    text += "하세요";
    return text;
  }

  //ES6
  const hi = (text) => {
    text += "하세요";
    return text;
  };
  ```

  ```javascript
  function getName(name) {
    return name;
  }

  //ES6
  const hi = (name) => {
    return name;
  };
  const hi = (name) => name;
  ```
