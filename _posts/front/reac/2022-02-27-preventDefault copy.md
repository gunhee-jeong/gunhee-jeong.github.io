<span style="color:red">npm run start</span>를 하는 과정에서 위의 사진과 같은 에러를 마주했다.

구글링을 하다보니 나의 <u>노드 버전이 높아서 발생하는 문제</u>인듯하였다.  
문제 해결을 위한 진행방향은 아래와 같다.

- 현재 노드의 버전 살펴보기

```bash
node -v
```

- 사용 가능한 노드 버전 확인하기

```bash
brew search node
```

- 현재 버전과의 연결 해제

```bash
brew unlink node
```

- Node 16버전 설치

```bash
brew install node@16
```

- 설치된 버전과의 연결

```bash
brew link node@16
```
